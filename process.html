Arbetsprocess

Här beksrivs arbetsprocessen för utveckling och publicering av webbplatsen.

Webbprojektet finns tillgängligt i ett publikt repo på GitHub: https://github.com/Safe-Pizza/DT211G_Lab1_Vite.git

Utveckling

GitHub repot klonas till din lokala dator. NodeJS används för att automatisera uppgifter, paketen finns i en package.json som återställs via kommandot nmp install. All utveckling och testning av webbplatsen sker i branchen dev. För skapa en demowebbplatsen under utveckligen använder du kommandot npm run dev, då får du en länk där du kan se webbplatsen lokalt i din webbläsare. När du sedan vill testköra webbplatsen innan publicering använder du kommandot npm run build, då byggs webbplatsen och en katalog för distribution skapas (dist), och sedan npm run preview för att få en lokal länk till den färdiga webbplatsen. Fungerar webbplatsen som tänk är du redo att gå vidare med publicering som beskrivs under rubriken Publicering. Om inte så raderar du dist-katalogen och fortsätter utveckligen i dev-branchen.
