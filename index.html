Startsida

Denna webbplats är en laboration i kursen DT211G - Frontend-baserad webbutveckling i programmet Webbutveckling hos Mittuniversitetet.

Mål och syfte

Laborationen ska ge inblick i hur NodeJs kan användas för att underlätta utveckling av webbplats. Lära sig automatisera uppgifter med något kommandobaserat verktyg i NodeJs. Lära sig skapa en automatiserad utvecklingsmiljö baserad på NodeJs och Git. Lära sig automatisera transpilering och minifiering av CSS, HTML och JavaScript samt automatiskt skapa optimerade grafikfiler.

Uppgift

Uppgiften består av flera delar som framgår i listan nedan:

  • Git och github ska användas för versionshantering,
  • NodeJs ska användas för att automatisera uppgifter i webbprojektet,
  • Fullt responsiv demowebbplats med tre undersidor skapas med HTML, CSS och JavaScript,
  • Webbplatsen ska publiceras till webbhost ocu automatiskt publiceras vid uppdatering av Github-repot,
  • Arbetsprocessen för utveckling och publicering ska beskrivas
